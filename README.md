\# Wi-Fi Presence Detector



A simple project that detects human presence and motion using Wi-Fi RSSI signals on a Windows laptop.



\## Features

\- Collect RSSI data with labels (`empty`, `occupied`, `moving`)

\- Train a RandomForest model

\- Run live detection in real time

\- Visualize RSSI waves



\## Requirements

\- Windows laptop with Wi-Fi

\- Python 3.9+
